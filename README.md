## What is MindForge?

MindForge is an Android app built for CS undergrads and self-taught developers who want to level up beyond tutorials. Instead of passive learning, it throws you into timed daily sessions with real problems across 6 skill domains — tracking your XP, accuracy, and streaks as you grind.

---

## Skill Domains

| Domain | Topics |
|:-------|:-------|
| ⚔️ **DSA** | Arrays, trees, graphs, DP, sorting — killer problems |
| 🧩 **Logic** | Puzzles, reasoning, probability, brain teasers |
| 🤖 **ML & AI** | Machine learning, neural nets, math behind AI |
| 🗄️ **DB & Systems** | Databases, OS, distributed systems, indexing |
| 🤝 **Negotiation** | Salary, deals, influence, high-stakes communication |
| 🧠 **Mind Control** | Focus, habits, cognitive biases, mental models |

---

## Features

- **Daily Sessions** — 30-minute timed sessions with 6 challenges per day
- **Streak Tracking** — stay consistent or lose your streak
- **XP & Leveling** — earn XP for every correct answer
- **Accuracy Stats** — track how sharp you really are
- **6 Skill Domains** — 100+ questions across CS and soft skills
- **Dark UI** — built for late-night grind sessions

## Install

1. Go to [**Releases**](https://github.com/Darkus39/mindforge/releases/latest)
2. Download `app-debug.apk`
3. On your Android phone → **Settings → Install unknown apps → Allow**
4. Open the APK and install

> Requires Android 7.0+

---

## Build from Source

```bash
# Prerequisites: Node.js, Cordova, Android SDK, JDK 17

npm install -g cordova
git clone https://github.com/Darkus39/mindforge.git
cd mindforge
cordova platform add android@12
cordova build android
```

APK output:
```
platforms/android/app/build/outputs/apk/debug/app-debug.apk
```
